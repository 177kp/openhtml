openHTML

A collaborative research project aimed at designing better tools and practices for learning web development.

Editor Research Team Contact

Editor

We've developed a web editor called openHTML Editor to helps beginners jump into coding in HTML and CSS.

Built on a fork of JSBin, we've added W3C validation, a coding session replayer, group features for classes and workshops, and more.

We're continuing to explore new features that will make their way into Mozilla Thimble.

Research

We're also conducting research on how people learn web development.

Drawing from the computing education research literature, we've studied web development novices in the lab, in workshops, and online. A few of our papers are listed here.

This work is supported by NSF grant CNS-1339344.

Learning web development: Challenges at an earlier stage of computing education, ICER 2011

Towards a taxonomy of errors in HTML and CSS, ICER 2013

openHTML: Designing a transitional web editor for novices, CHI 2013
